# tmux

## commands
```bash
# list current tmux session
tmux ls

# detach a tmux session
ctrl+b and d

# attach to an existing session
tmux attach-session -t <session-number>

# kill a session
tmux kill-session -t <session-number>

# rename session
tmux rename-session -t <session-number> <new-name>

# open a new window
ctrl+b and c

# open a new tab in the same window
ctrl+b and %

# open a new tab horizontally
ctrl+b and "

```
